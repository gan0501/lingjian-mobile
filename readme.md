## 核心功能模块
| 模块 | 说明 |
|------|------|
| **找项目** | 地图展示规划工程、土地拍卖、招投标、采购公告 |
| **找建企** | 地图展示招标造价、设计勘察、施工安装、监理检测企业 |
| **找厂家** | 地图展示劳务班组、材料厂家、机械设备、商务服务供应商 |
| **找资源** | 建筑规范、标准图集、信息价资料查阅 |
| **AI工具** | AI写标书、桩基承载力计算、3D白模生成、CAD查看、水印相机、证件照 |
| **AI跟进** | 4智能体协同跟进项目（对话、笔记、跟进、总结） |
