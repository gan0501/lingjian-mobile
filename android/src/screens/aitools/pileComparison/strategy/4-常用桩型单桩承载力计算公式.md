
# 常见桩型的单桩承载力计算公式

## 一、单桩竖向抗压承载力计算公式

### 1. 通用单桩竖向抗压极限承载力（包括管桩、空心方桩、实心方桩、钻孔灌注桩）
**公式**：$Q_{uk}=Q_{sk}+Q_{pk}=u\sum q_{sik}l_{i}+q_{pk}A_{p}$
**参数说明**：
- $Q_{uk}$：单桩竖向抗压极限承载力（kN）
- $u$：桩身周长（m），圆形桩$u=\pi d$，方形桩$u=4b$
- $q_{sik}$：第i层土的桩侧极限摩阻力标准值（kPa）
- $l_i$：第i层土中桩身长度（m）
- $q_{pk}$：桩端极限端阻力标准值（kPa）
- $A_p$：桩端截面积（m²）
**常见规格**
- 管桩（预制圆形桩）：直径400mm、500mm、600mm
- 空心方桩（预制方形桩）：边长400mm、450mm、500mm
- 实心方桩（预制方形桩）：边长400mm、450mm、500mm
- 钻孔灌注桩（圆形桩）：直径600mm、650mm、700mm、800mm、900mm、1000mm

### 2. 按桩身强度验算（控制值）
**公式**：$Q_{a}\leq0.85\psi_{c}f_{c}A_{p}$
**参数说明**：
- $\psi_{c}$：基桩成桩工艺系数（钻孔灌注桩取0.7~0.8，预制桩取0.85~0.95）
- $f_{c}$：混凝土轴心抗压强度设计值（kPa）

### 3. CFG桩单桩竖向抗压承载力特征值
**公式**：$R_a=U_p\sum q_{sai}l_i+\alpha q_pA_p$
**参数说明**：
- $U_p$：桩周长（m）
- $q_{sai}$：桩周第i层土的侧阻力特征值（kPa）
- $\alpha$：桩端土地基承载力折减系数
- $q_p$：桩端端阻力特征值（kPa）

### 4. 钢管桩承载力修正
**公式**：$R_a=\psi_c\cdot R_{a0}$
**参数说明**：
- $\psi_c$：钢管桩闭塞效应系数（闭口桩取1.0，开口桩取0.8~0.9）

### 5. 嵌岩桩单桩轴向受压容许承载力
**公式**：$[P]=c_1U\sum\tau_i l_i+c_2A\sigma_R$
**参数说明**：
- $[P]$：单桩轴向受压容许承载力（kN）
- $c_1$、$c_2$：根据清孔情况、岩石破碎程度等因素确定的系数
- $U$：桩嵌入持力层的横截面周长（m）
- $\tau_i$：第i层土与桩壁的极限摩阻力（kPa）
- $l_i$：第i层土的厚度（m）
- $A$：桩底横截面面积（m²）
- $\sigma_R$：天然湿度的岩石单轴极限抗压强度（kPa）

### 6. 挤扩支盘桩单桩竖向极限承载力
**公式**：$Q_u=u\sum q_{si}L_i+\sum q_{pj}A_{pj}+q_pA_p$
**参数说明**：
- $L_i$：桩穿越第i层土折减盘高的有效厚度
- $q_{pj}$：承力盘端阻力标准值（kPa）
- $A_{pj}$：承力盘面积（m²）

### 7. 异型桩竖向抗压极限承载力（依据JGJ/T 405-2017）
**公式**：$Q_{uk}=Q_{sk}+Q_{pk}=β{c} u\sum q_{sik}l_{i}+q_{pk}A_{p}$
**参数说明**：
- $β{c}$：桩身侧阻力调整系数（取值范围：1.1~1.3）
- $Q_{uk}$：单桩竖向抗压极限承载力（kN）
- $u$：桩身周长（m），圆形桩$u=\pi d$，方形桩$u=4b$
- $q_{sik}$：第i层土的桩侧极限摩阻力标准值（kPa）
- $l_i$：第i层土中桩身长度（m）
- $q_{pk}$：桩端极限端阻力标准值（kPa）
- $A_p$：桩端截面积（m²）
**常见规格**
- 异型空心方桩（预制方形桩）：边长400mm、450mm、500mm
- 异型实心方桩（预制方形桩）：边长400mm、450mm、500mm

### 8.静钻根植桩抗压极限承载力（依据DB 33/T 1134-2017、T/CECS 738-2020）

**公式**：$Q_{uk} = \sum u_i q_{sik} L_i + q_{pk} A_p$$
**参数说明**：
- $u_i$：桩身周长，竹节桩按节外径计算，其他类型按桩外径计算
- $q_{sik}$：桩周第i层土的极限侧阻力标准值，参照岩土工程勘察报告提供的预制桩极限侧阻力标准值取值
- $L_i$：第i层土的厚度
- $q_{pk}$：极限桩端阻力标准值，参照岩土工程勘察报告中的预制桩极限端阻力标准值取值并乘以相应折减系数
- $A_p$：桩端截面面积，不扩底时取钻孔底部截面积，扩底时取扩底部截面积
**静钻根植桩承载力计算关键问题**
1. **桩侧摩阻力计算**：桩身周长是否计入外围水泥土尚不统一，建议采用钻孔直径计算静钻根植桩侧摩阻力
2. **桩端阻力计算**：建议采用桩端扩底截面积计算桩端阻力
3. **修正系数**：需根据实测数据确定桩侧阻力修正系数和桩端阻力修正系数
**静钻根植桩技术特点**
- **工艺流程**：钻孔→扩底→注浆→植桩
- **适用桩径**：500-1200mm
- **最大深度**：80m
- **承载力提升**：较传统桩基提升45%以上
- **环保效益**：泥浆减排70%，无挤土效应，低噪声

### 9. 劲性芯复合桩技术规程计算公式

**公式**：$Q_{uk} = u_p\sum\xi_{si}q_{sia}L_i + \xi_p q_{pa}(A_p - A_c) + a\xi_p q_{pa}A_c$$
**参数说明**：
- $u_p$：外层水泥土搅拌桩的桩周长
- $\xi_{si}$：桩身复合段的侧摩阻力调整系数
- $q_{sia}$：桩复合段第i层土的极限侧阻力标准值
- $\xi_p$：桩端阻力调整系数
- $q_{pa}$：桩端阻力标准值
- $A_p$：桩端截面面积
- $A_c$：内芯截面面积
- $a$：经验系数

## 二、单桩竖向抗拔承载力计算公式

### 1. 通用抗拔承载力公式
**公式**：$T_{uk}=\sum\lambda_{i}q_{sik}ul_{i}+G_{p}$
**参数说明**：
- $T_{uk}$：单桩竖向抗拔极限承载力（kN）
- $\lambda_{i}$：第i层土的抗拔系数（黏性土取0.7~0.8，砂土取0.5~0.7）
- $G_{p}$：桩身自重（kN），$G_{p}=\gamma_{c}A_{p}l$（$\gamma_{c}$为混凝土重度，取25kN/m³）

### 2. 挤扩支盘桩单桩抗拔极限承载力
**公式**：$U_u=u\sum\lambda_i q_{si}L_i+\sum q_{pj}A_{pj}$
**参数说明**：
- $U_u$：挤扩支盘桩单桩抗拔极限承载力标准值（kN）
- $\lambda_i$：桩周第i层土的侧阻力折减系数

## 三、设计值计算

**公式**：$Q_a=\frac{Q_{uk}}{K}$（抗压）或 $T_a=\frac{T_{uk}}{K}$（抗拔）
**参数说明**：
- $K$：安全系数（抗压一般取2，抗拔一般取2.5）

## 四、主要规范依据

1. 《建筑桩基技术规范》（JGJ94-2008）
2. 《建筑地基基础设计规范》（GB50007-2011）
3. 《公路桥涵地基与基础设计规范》（JTG3363-2019）
4. 《复合地基技术规范》(GB/T50783-2012)
5. 《建筑地基处理技术规范》(JGJ79-2012)


以上公式适用于不同桩型的承载力计算，实际应用时应根据具体工程条件、地质情况和相关规范要求进行参数选取和计算。
