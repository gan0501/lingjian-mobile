### 一、基本原理：什么土更适合做持力层？

选择持力层的总体目标是“**承载力高、压缩性低、厚度大、分布均匀**”。具体应遵循以下原则：

1. 力学性能优良（硬）
- **高强度：** 优先选择标准贯入试验锤击数 $N$ 值较大、物理力学指标（如内摩擦角 $\phi$、黏聚力 $c$、压缩模量 $E_s$）较高的土层。
- **低压缩性：** 持力层在荷载作用下自身变形要小，以控制建筑物的沉降。

2. 厚度足够（厚）
- 桩端进入持力层需要有一定的**深度**。
- 对于粘性土、粉土，不宜小于 $2d$（$d$ 为桩径）。
- 对于砂类土，不宜小于 $1.5d$。
- 对于碎石类土，不宜小于 $1d$。
- **注意：** 如果持力层厚度不够，桩端可能会刺穿持力层进入下卧软弱层，造成破坏。

3. 层位稳定（稳）
- 持力层在水平方向上的分布应**均匀**，层面坡度不宜过陡（一般要求坡度 $<10\%$）。
- 避免桩基一部分落在岩面上，另一部分落在土层中（即“一脚软一脚硬”），这会导致不均匀沉降和应力集中。

4. 排除不良地质（净）
- **避开软弱下卧层：** 持力层下方不应存在未胶结的断层破碎带、溶洞、暗浜或高压缩性软土层。
- **避开液化层：** 在地震区，应避开可液化的砂土或粉土层。

**说明.** 常见持力层类型对比
在实际工程中，不同土层作为持力层的效果差异很大：
| 土层类型 | 作为持力层的评价 | 注意事项 |
| :--- | :--- | :--- |
| **基岩** (花岗岩、石灰岩等) | **最优**。承载力极高，几乎无压缩。 | 需注意风化程度（全风化、强风化、中风化、微风化）。通常以**中风化或微风化**为佳。 |
| **密实砾砂/卵石层** | **优良**。承载力高，工程性质稳定。 | 桩端进入该层一定深度即可提供巨大阻力。 |
| **硬塑~坚硬状态粘土** | **良好**。承载力较高。 | 需注意是否存在膨胀土特性或遇水软化的可能。 |
| **中密状态粉土/砂土** | **一般**。可用作多层建筑持力层。 | 承载力相对较低，需验算沉降。 |
| **松散填土/淤泥质土** | **严禁**。属于典型的软弱土。 | 这类土通常只能作为桩身穿越的土层，绝不能作为端承持力层。 |

### 二、确定持力层的思路

在实际操作中，确定持力层是一个结合地质报告和设计计算的过程：

1. 仔细阅读地质勘察报告
- 查看**钻孔柱状图**和**工程地质剖面图**。
- 找出标贯击数 $N$ 值突变（突然变大）的土层，这通常意味着进入了硬层。
- 查看土层的物理力学性质指标表。

2. 初步筛选
- 根据建筑物的荷载大小，排除掉承载力明显不足的软弱土层。
- 排除掉厚度太薄（小于桩径）的夹层。
- 排除掉层面起伏太大的土层。

3. 计算与验算
- **计算单桩承载力：** 利用规范公式，计算桩侧摩阻力 $Q_s$ 和桩端阻力 $Q_p$。
    $$Q_u = Q_s + Q_p$$
    确保总承载力满足设计要求。
- **软弱下卧层验算：** 如果持力层下方不远处有软弱土层，必须验算持力层是否会将荷载扩散刺穿到下卧层。
- **沉降计算：** 确认该持力层能将建筑物的沉降控制在允许范围内。

4. 经济性与施工可行性
- **桩长控制：** 持力层埋藏太深会导致桩长过长，造价剧增；埋藏太浅（如直接在地表的岩石）可能需要爆破，增加施工难度。
- **穿越能力：** 桩必须能够顺利穿越上覆土层到达持力层。例如，如果有很厚的密实砂层，普通的静压桩可能无法穿过，需改用钻孔灌注桩。

**注意** 特殊情况处理
- **岩溶地区：** 持力层必须座落在完整的岩盘上，且桩底以下一定深度（如3-5倍桩径）范围内无溶洞、裂隙。
- **斜坡场地：** 桩端应嵌入倾斜的基岩中，确保桩底全截面嵌入岩石，防止滑移。
- **摩擦型桩：** 如果没有坚硬的持力层（如深厚的软土地区），则不强调“端承”，主要依靠长桩的侧摩阻力（摩擦桩），此时所谓的“持力层”概念弱化，主要关注桩长。

   - 排除指标：设计参数表中未提供桩端承载力参数的土层；
   - 输出结果：原表提供了桩端承载力参数的为xx层、xx层（存入候选池），其余视为不适宜土层。

2. 劣质过滤维度（排除不适用土层）
   - 排除指标：土层类型（软弱土/液化土）、土层状态（松散/流塑）、有效厚度（＜桩径倍数阈值）；
   - 输出结果：剔除XX层（如①填土、②淤泥质土），剩余XX层为候选池。

3. 承载力优先级维度（锁定核心候选）
   - 排序指标：极限端阻q_pk（核心）、单位长度总阻力（修正）；
   - 输出结果：按q_pk从高到低排序，Top3候选层为③-1、④、⑤（示例）。

4. 安全规范约束维度（确保合规）
   - 校验指标：下卧层承载力、沉降量、桩端埋深、进入持力层深度；
   - 输出结果：合规层为③-1、④（⑤因下卧层q_pk不达标剔除）。

【方案总结】
1. 最优解（⑤圆砾）：承载力最大、合规性强，为当前地质+设计参数下的最佳选择，尤其在高承载力需求场景，推荐优先采用，适用于钻孔桩（大荷载），单桩承载力通常在2500KN以上；
