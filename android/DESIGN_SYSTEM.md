# 领建AI - 设计规范

## 一、颜色

### 1.1 背景

| 名称 | 色值 | 用途 |
|------|------|------|
| **主背景** | `#000000` | 非首页页面 |
| **首页背景** | 渐变 `#80011A` → `#000000` | 首页专用 |
| **表面色** | `#0A0E14` | 卡片背景 |
| **次级表面** | `#1A222D` | 弹窗内层、卡片内层 |
| **弹窗背景** | `#FFFFFF` | 底部弹窗、中心弹窗 |

### 1.2 文字

| 名称 | 色值 | 用途 |
|------|------|------|
| **主色** | `#F0F4F8` | 标题、重要文字 |
| **次色** | `#A0AEC0` | 描述、次要文字 |
| **三级** | `#718096` | 辅助文字 |
| **弹窗文字** | `#1A1A1A` | 白色弹窗内文字 |

### 1.3 强调

| 名称 | 色值 | 用途 |
|------|------|------|
| **主强调** | `#C084FC` | 链接、高亮 |
| **按钮主色** | `#000000` | 确定按钮、主要按钮 |
| **成功** | `#10B981` | 成功状态 |
| **警告** | `#F59E0B` | 警告状态 |
| **错误** | `#EF4444` | 错误状态 |

### 1.4 玻璃效果

| 名称 | 色值 | 用途 |
|------|------|------|
| **边框** | `rgba(255,255,255,0.12)` | 卡片边框 |
| **高光** | `rgba(255,255,255,0.3)` | 顶部高光线 |
| **遮罩** | `rgba(0,0,0,0.7)` | 弹窗遮罩 |

## 五、弹窗

### 5.1 侧边弹窗

| 属性 | 值 |
|------|-----|
| 位置 | 右侧滑入 |
| 宽度 | 80% |
| 背景 | `rgba(28, 20, 45, 0.98)` |
| 边框 | 左边框 `rgba(192, 132, 252, 0.3)` |

**用途**：个人信息、消息中心、找项目、找建企、找厂家、我的标书、对比报告

### 5.2 底部弹窗

| 属性 | 值 |
|------|-----|
| 位置 | 底部滑入 |
| 背景 | `#FFFFFF` |
| 圆角 | 顶部 20px |

**用途**：内容展示、表单填写、筛选条件

### 5.3 中心弹窗

| 属性 | 值 |
|------|-----|
| 位置 | 居中 |
| 背景 | `#FFFFFF` |
| 圆角 | 16px |
| 遮罩 | `rgba(0, 0, 0, 0.7)` |

**用途**：确认、取消等操作

### 5.4 轻提示

| 属性 | 值 |
|------|-----|
| 位置 | 顶部 |
| 时长 | 3秒自动消失 |
| 圆角 | 12px |

**用途**：结果提示、搜索结果数量
